Syllabus Content  Download

Unit-1: Introduction

Introduction: Objective of Operating Systems: Definition, Evolution of Operating systems, Types of Operating Systems, OS Services, System Calls, OS structure: Layered, Monolithic, Microkernel Operating Systems

Unit-2:  Process Concepts

Overview , Process Relationship , Process states , Process State transitions ,Process Control Block ,Context switching,Threads,Concept of multithreads, Advantages and Type of threads

Unit-3:  Process Scheduling

Scheduling objectives, Types of Schedulers, Scheduling criteria, Scheduling algorithms, Multiprocessor scheduling, Thread Scheduling Performance Analysis of the scheduling.

Unit-4:  Interprocess Communication

Race Conditions, Critical Section, Mutual Exclusion, Hardware Solution, Peterson’s Solution, The Producer Consumer Problem, Semaphores, Event Counters, Message Passing, and Classical IPC Problems: Reader’s & Writer Problem, Dinning Philosopher Problem, Monitors

Unit-5:  Deadlock

Definition, Deadlock characteristics, Deadlock Prevention, Deadlock Avoidance banker’s algorithm, Deadlock detection and Recovery.

Unit-6:  Basic Memory Management

Background of Memory management, Swapping, Contiguous Memory allocation, Paging, Structure of Page Table, segmentation

Unit-7:  Virtual Memory

Basics of Virtual Memory ,Demand paging, Page Replacement policies : Optimal (OPT) , First in First Out (FIFO), Second Chance (SC), Not recently used (NRU) and Least Recently used (LRU)

Unit-7:  Storage Structure & I/O Management

Overview of Secondary Storage Structure, Disk structure ,Disk scheduling algorithm, RAID Structure ,Principles of I/O Hardware :I/O devices, Device controllers , Direct memory access Principles of I/O Software: Goals of Interrupt handlers, Device drivers, Device independent I/O software

Unit-7: File Management

File concept, Access methods, File types, File operation, Directory structure, File System structure, Allocation methods (contiguous,linked,indexed), Freespace management (bit vector, linked list, grouping),directory implementation (linear list, hash table).

Unit-7: Basics of Unix/Linux Operating System

Development Of Unix/Linux, Role & Function Of Kernel, Directory Structure, System Administration
