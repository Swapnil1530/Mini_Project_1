Syllabus Content  Download

Unit-1: Introduction

Purpose & Applications, Database Architecture, Type of DBMS, Data Model, Users Role & Responsibility, Transaction Management

Unit-2:  Relational Model

Structure of Relational Database, Database Schema, Keys, Relational Algebra Operations, Relational Algebra Query Language

Unit-3: SQL

Overview of the SQL Query Language, SQL Data Definition, Basic Structure of SQL Queries, Integrity Constraint, Additional Basic Operations, Set Operations, Null Values, Aggregate Functions, Nested Sub queries, Modification of the Database, Join Expressions, Aggregation Features

Unit-4:  E-R Model

: Overview, Constraints, E-R Diagrams, E-R Design Issues, Advance E-R Model

Unit-5:  Database Design

Features of Good Database Design, Redundancy, Functional Dependency, Decomposition, Normalizations, Normal Forms

Unit-6: Query Processing & Optimization

Overview, Estimating of Query Cost, Evaluation of Expressions, Evaluation Plan, Selection Operation, Sorting, Join Operations, Indexing, Ordered Indices, B+ Tree Index, Static and Dynamic Hashing

Unit-7:  Transaction Management

Concept, Transaction Model, ACID Property, Serializability, Transaction Protocols, Deadlock Handling, Recovery

Unit-8:  Advance SQL

Views, Index, Transactions, Authorization, Functions and Procedures, Triggers, Cursor

Unit-7:  Case Study

Postgre SQL, MySQL

Course Outcome

After learning the course the students should be able to:

  • Understand the business data and relation between data
  • Use of Technology to store the large data
  • Efficiently usage of data in short response time
  • Access control over the data
  • Design of database for real life project
